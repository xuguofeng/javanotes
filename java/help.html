本套 Java 零基础笔记主要是作者结合日常学习笔记、实际开发案例整理而成，其中不乏网络论坛中的资源和论点，在此表示感谢。


全套笔记主要包括：Java 核心技术、Java web 开发（Servlet、Jsp、jsp）、MVC 分层思想及 SSH 框架、数据库基础以及HTML、CSS、JavaScript等。


现在正在更新的笔记是 Java 核心技术内容，在此部分，我们将接触到 Java 语法、Java 面向对象编程、Java 多线程技术、JavaIO 技术、JavaGUI 编程、Java 网络编程等。如果您只是想了解一些基础的编程知识或者只是想在 Java 桌面程序开发方面发展，那么您只需要阅读这部分的文章和少量的数据库知识即可。


如果您想做网站、做企业级开发，那么您就需要多学习一些 Java Servlet API、MVC 分层思想和分层框架，当然，数据库、HTML 相关的技术也是需要的。


现在只发布了 Java 语法和 Java 面向对象编程两部分，后面的内容会陆续进行更新。


在更新 Java 笔记的过程中，我们会不定时的更新 Linux 服务器搭建的相关知识，包括：Linux 基础命令、Linux Shell 编程基础、服务器搭建和服务器集群架构等。
