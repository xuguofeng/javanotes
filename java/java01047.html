一、GUI 概述

1、 学习目标

从本节开始，我们将开始学习 Java 的 GUI 编程，GUI 全称 Graphical User Interface，即“图形用户接口”，也就是我们常说的图形界面软件。与 CLI （Command Line User Interface） 操作方式相比，图形界面操作简单、用户体验更好。

目标就是完成“简单资源管理器”和“坦克大战”的程序。

“简单资源管理器”可以实现查看系统分区、文件、获取文件信息，还可以导出当前所在文件夹的文件树。


“坦克大战”可以实现坦克的运动、对战，显示用户成绩。


2、 GUI 概述

Java 为 GUI 提供的对象都在 java.awt 和 javax.swing

awt：Abstract Window Toolket(抽象窗口工具包)，需要调用本地系统方法实现功能，属于重量级控件。

swing 在 awt 基础上，建立的一套图形界面系统，其中提供了更多的组件，而且完全有 java 实现。增强了移植性，属于轻量级控件。

二、AWT 组件、容器和窗体

1、 Component 类
java.awt;

public abstract class Component implements ImageObserver, MenuContainer {}

具有图形表示能力的类，可在屏幕上显示与用户进行交互。例如图形用户界面中的窗口、弹出框、按钮、复选框和滚动条等。


有以下主要字段 

public static final float TOP_ALIGNMENT
	指定组件顶部对齐

public static final float BOTTOM_ALIGNMENT
	指定组件底部对齐

public static final float LEFT_ALIGNMENT
	指定组件左对齐

public static final float RIGHT_ALIGNMENT
	指定组件右对齐

public static final float CENTER_ALIGNMENT
	指定组件居中对齐

有以下主要方法 

public int getWidth()
	返回组件的当前宽度

public int getHeight()
	返回组件的当前高度

public int getX()
	返回组件原点（左上角）相对于屏幕左上角的x坐标

public int getY()
	返回组件原点（左上角）相对于屏幕左上角的y坐标

public void setSize(int width, int height)
	调整组件的大小

public Dimension getSize()
	以Dimension对象的形式返回组件的大小

public void setLocation(int x, int y)
	将组件移到新位置

public Point getLocation()
	获取组件位置，形式是指定组件左上角的点

public void setEnabled(boolean b)
	启用或禁用组件

public void setVisible(boolean b)
	显示或隐藏组件

public void add(PopupMenu popup)
	向组件添加指定弹出菜单

public void remove(MenuComponent popup)
	从组件移除指定弹出菜单

public void addComponentListener(ComponentListener l)
	添加组件侦听器，以接收发自组件的事件

public void removeComponentListener(ComponentListener l)
	移除组件侦听器，不再接收发自组件的组件事件

public void addFocusListener(FocusListener l)
	添加焦点侦听器，当此组件获得输入焦点时能够接收发自此组件的焦点事件

public void removeFocusListener(FocusListener l)
	移除指定焦点侦听器，不再接收发自此组件的焦点事件

public void addKeyListener(KeyListener l)
	添加按键侦听器，接收发自此组件的按键事件

public void removeKeyListener(KeyListener l)
	移除按键侦听器，不再接收发自此组件的按键事件

public void addMouseListener(MouseListener l)
	添加鼠标侦听器，以接收发自此组件的鼠标事件

public void removeMouseListener(MouseListener l)
	移除鼠标侦听器，不再接收发自此组件的鼠标事件

public boolean contains(int x, int y)
	检查组件是否“包含”指定的点，其中x和y是相对于此组件的坐标系定义的

public void setBackground(Color c)
	设置组件背景色，可以使用setBackground(Color.green)设置背景色

public Color getBackground()
	获取组件背景色

public void setFont(Font f)
	设置组件字体

public Font getFont()
	获取组件字体

public void setForeground(Color c)
	设置组件前景色

public Color getForeground()
	获取组件前景色

public void setCursor(Cursor cursor)
	设置光标图像，可以使用setCursor(new Cursor(Cursor.HAND_CURSOR))方式设置为手形

public Cursor getCursor()
	获取组件中光标集合

public Graphics getGraphics()
	为组件创建一个图形上下文

public void paint(Graphics g)
	绘制此组件

public void paintAll(Graphics g)
	绘制此组件及其所有子组件

public void repaint()
	重绘此组件

* 继承此类的组件也都拥有这些方法，所以在介绍子类时就不再介绍这些方法了

2、 Container 类
java.awt;

public class Container extends Component {}

添加到容器中的组件放在一个列表中。列表的顺序将定义组件在容器内的正向堆栈顺序。


有以下主要方法 

public Component add(Component comp)
	将指定组件追加到此容器的尾部
	如果已经将某个组件添加到显示的容器中，则必须在此容器上调用validate，以显示新的组件
	如果添加多个组件，那么可以在添加所有组件之后，通过只调用一次validate来提高效率

public Component getComponent(int n)
	获取此容器中第n个组件

public int getComponentCount()
	获取此容器中组件数

public Component[] getComponents()
	获取此容器中所有组件

public void remove(Component comp)
	从此容器中移除指定组件

public void remove(int index)
	从此容器中移除index指定的组件

public void removeAll()
	从此容器中移除所有组件

public void validate()
	验证此容器及其所有子组件，以便在添加新组件时可以显示出来

public void setLayout(LayoutManager mgr)
	设置此容器的布局管理器

3、 Window 类
java.awt;

public class Window extends Container implements Accessible {}

Window 是一个没有边界、菜单栏的顶层窗口。默认布局是 BorderLayout


构造方法 

public Window(Frame owner)
	创建一个新的、不可见的窗口，使用指定Frame作为其所有者

public Window(Window owner)
	创建一个新的、不可见的窗口，使用指定Window作为其所有者

主要方法 

public void addWindowFocusListener(WindowFocusListener l)
	添加窗口焦点侦听器，从此窗口接收窗口事件

public void removeWindowFocusListener(WindowFocusListener l)
	移除窗口焦点侦听器，不再从此窗口接收窗口事件

public void addWindowListener(WindowListener l)
	添加窗口侦听器，从此窗口接收窗口事件

public void removeWindowListener(WindowListener l)
	移除窗口侦听器，不再从此窗口接收窗口事件

public void dispose()
	释放由此Window及其拥有的所有子组件使用屏幕资源

public void pack()
	调整此窗口的大小，以适合其子组件的首选大小和布局

public void setIconImage(Image image)
	设置窗口图标显示的图像

4、 Frame 类
java.awt;

public class Frame extends Window implements MenuContainer {}

带有标题和边框的顶层窗口

默认布局为BorderLayout

窗体的大小包括被边框指定的所有区域。边框区域的尺寸可以使用getInsets方法获得


构造方法 

public Frame() throws HeadlessException
	创建一个不可见的Frame窗体

public Frame(String title) throws HeadlessException
	创建一个新的、不可见的、具有指定标题的Frame窗体

主要方法 

public void setMenuBar(MenuBar mb)
	设置菜单栏

public MenuBar getMenuBar()
	获取菜单栏

public void remove(MenuComponent m)
	从窗体移除指定菜单栏

public void setTitle(String title)
	设置标题

public String getTitle()
	获得窗体标题

5、 第一个 FrameStart 程序
/**
 * 第一个Frame演示程序
 * 主要演示Frame类的setSize、setLocation、setTitle、setCursor、setBackground、setVisible
 * 以及添加WindowListener以实现关闭程序
 * 
 * @author soft01
 *
 */
@SuppressWarnings("serial")
public class FrameStart extends Frame {

	public FrameStart() throws HeadlessException {
		this(null);
	}

	public FrameStart(String title) throws HeadlessException {
		// 设置窗体大小
		// 第一个参数是宽
		// 第二个参数是高
		this.setSize(500, 400);

		// 设置窗体位置居中
		this.setLocation(
				(Toolkit.getDefaultToolkit().getScreenSize().width - 500) / 2,
				(Toolkit.getDefaultToolkit().getScreenSize().height - 400) / 2);

		// 为窗体设置标题
		this.setTitle(title);

		// 设置鼠标移入窗体时显示为手形
		this.setCursor(new Cursor(Cursor.HAND_CURSOR));

		// 设置窗体背景为绿色
		this.setBackground(Color.green);

		// 禁用最大化按钮
		this.setResizable(false);

		// 设置窗口图标
		this.setIconImage(new ImageIcon("images/wx.jpg").getImage());

		// 窗体关闭按钮点击监听器
		// 实现关闭按钮点击时退出程序
		// 因为默认情况下关闭按钮点击时什么都不做
		this.addWindowListener(new FrameCloseListener());

		// 设置窗体可见
		this.setVisible(true);
	}

	/**
	 * 内部类继承WindowAdapter类，重写windowClosing方法实现窗体关闭退出程序
	 * 
	 * @author soft01
	 *
	 */
	private class FrameCloseListener extends WindowAdapter {
		@Override
		public void windowClosing(WindowEvent e) {
			// 程序退出
			// 返回值给JVM，JVM再返回给操作系统
			System.exit(0);
		}
	}

	public static void main(String[] args) {
		// new FrameStart();
		new FrameStart("第一个演示窗体");
	}
}

运行程序可以显示窗口
