一、Hashtable 类

java.util;

public class Hashtable<K,V> extends Dictionary<K,V> implements Map<K,V>, Cloneable, Serializable {}

此类使用哈希表将键映射到相应的值，任何非 null 对象都可以用作键或值。

为了成功地在哈希表中存储和获取对象，用作键的对象必须实现 hashCode 方法和 equals 方法。

从 Java 2 平台 v1.2 起，此类被改进以实现 Map 接口，成为集合框架的一个成员，Hashtable 是同步的。

HashMap 和 Hashtable 有以下共同点和区别：

（1）二者内部都是哈希表实现；

（2）前者是 1.2 出现的，是 Map 接口的实现类；后者是过时的 Dictionary 的子类；JDK 已经明确建议“新的实现应该实现 Map 接口，而不是扩展 Dictionary 类。”；

（3）前者不是同步的，多个线程操作需要实现外部同步；后者是同步的；

（4）前者允许 null 键、null 值，后者不允许 null 作为键值；

二、Properties 类

1、 概述
java.util;

public class Properties extends Hashtable<Object, Object> {}

表示一个持久的属性集，可以保存在流中或从流中加载，属性集中每个键及其对应值都是一个字符串。

一个属性集可以包含另一个属性集作为“默认值”。

load(Reader)/store(Writer, String) 方法在字符流中加载和存储属性。

loadFromXML(InputStream) 和 storeToXML(OutputStream, String, String) 方法按简单的 XML 格式加载和存储属性。默认使用 UTF-8 字符编码，当然也可以指定编码。XML 属性文档应具有以下 DOCTYPE 声明：

<!DOCTYPE properties SYSTEM "http://java.sun.com/dtd/properties.dtd">

2、 构造方法
public Properties()
	创建一个无默认值的空属性集。

public Properties( Properties defaults )
	创建一个带有指定默认值的空属性集。
3、 主要方法
public String getProperty(String key)
	用指定的键在此属性集中搜索属性值。

public String getProperty(String key, String defaultValue)
	用指定的键在属性集中搜索属性。如果在属性集中未找到该键，则接着递归检查默认属性集及其默认值。如果未找到属性，则此方法返回默认值变量。

public void list(PrintStream out)
	将属性集输出到指定的输出流。

public void list(PrintWriter out)
	将属性集输出到指定的输出流。

public void load(InputStream inStream) throws IOException
	从输入流中读取属性集（键和元素对）

public void load(Reader reader) throws IOException
	按面向行的格式从字符输入流中读取属性集（键和元素对）。

public Object setProperty(String key, String value)
	为某个键设置指定的值。

public void store(OutputStream out, String comments) throws IOException
	将属性集（键和元素对）写入输出流。

public void store(Writer writer, String comments) throws IOException
	将属性集（键和元素对）写入字符输出流。

public Set<String> stringPropertyNames()
	返回属性集中的键集。

三、集合数组工具类

1、 Collections 类
java.util;

public class Collections extends Object {}

此类完全由在 Collection 上进行操作或返回 Collection 的静态方法组成，具体方法如下： 

public static <T> boolean addAll(Collection<? super T> c, T... elements)
	将所有指定元素添加到指定集合。

public static <T> int binarySearch(List<? extends Comparable<? super T>> list, T key)
	使用二分法搜索指定集合，以获得指定对象。在调用之前，必须根据元素的自然顺序对集合进行升序排序（通过sort(List)方法）。

public static <T> int binarySearch(List<? extends T> list, T key, Comparator<? super T> c)
	使用二分法搜索指定集合，以获得指定对象。在调用之前，必须根据指定的比较器对集合进行升序排序（通过sort(List, Comparator)方法）。

public static <T> void copy(List<? super T> dest, List<? extends T> src)
	将所有元素从一个集合复制到另一个集合。

public static <T> void fill(List<? super T> list, T obj)
	使用指定元素替换指定集合中的所有元素。

public static <T extends Object & Comparable<? super T>> T max(Collection<? extends T> coll)
	根据元素的自然顺序，返回给定集合的最大元素。

public static <T> T max(Collection<? extends T> coll, Comparator<? super T> comp)
	根据指定比较器产生的顺序，返回给定集合的最大元素。

public static <T extends Object & Comparable<? super T>> T min(Collection<? extends T> coll)
	根据元素的自然顺序，返回给定集合的最小元素。

public static <T> T min(Collection<? extends T> coll, Comparator<? super T> comp)
	根据指定比较器产生的顺序，返回给定集合的最小元素。

public static <T> boolean replaceAll(List<T> list, T oldVal, T newVal)
	使用newVal替换集合中出现的所有oldVal。

public static <T> Comparator<T> reverseOrder()
	返回一个比较器，逆转实现了Comparable接口的对象集合的自然顺序。
	例如，假设 a 是一个字符串数组。那么：
	Arrays.sort(a, Collections.reverseOrder());
	将按照逆字典（字母）顺序对数组进行排序。

public static <T> Comparator<T> reverseOrder(Comparator<T> cmp)
	返回一个比较器，逆转指定比较器的顺序。

public static <T extends Comparable<? super T>> void sort(List<T> list)
	根据元素的自然顺序对指定集合按升序进行排序。集合中的所有元素都必须实现Comparable接口。

public static <T> void sort(List<T> list, Comparator<? super T> c)
	根据指定比较器产生的顺序对指定集合进行排序。此集合内的所有元素都必须可使用指定比较器相互比较。

public static <T> List<T> synchronizedList(List<T> list)
	返回指定集合支持的同步（线程安全的）集合。

public static <K,V> Map<K,V> synchronizedMap(Map<K,V> m)
	返回由指定映射支持的同步（线程安全的）映射。

public static <T> Set<T> synchronizedSet(Set<T> s)
	返回指定Set支持的同步（线程安全的）Set。
2、 Arrays 类
java.util;

public class Arrays extends Object {}

此类包含用来操作数组的各种方法，具体如下： 

public static <T> List<T> asList(T... a)
	返回一个受指定数组支持的固定大小的集合。

public static int binarySearch(int[] a, int key)
	使用二分法搜索指定的int型数组，以获得指定的值。

public static int binarySearch(int[] a, int fromIndex, int toIndex, int key)
	使用二分法搜索指定的int型数组，在指定范围内获得指定的值，必须在调用前对范围进行排序（通过sort(int[], int, int)方法）。

public static int binarySearch(Object[] a, int fromIndex, int toIndex, Object key)
	使用二分法搜索指定数组，在指定范围内获得指定对象，在调用前，必须根据元素的自然顺序对范围进行升序排序（通过sort(Object[], int, int)方法）。

public static int binarySearch(Object[] a, Object key)
	使用二分法搜索指定数组，以获得指定对象，在调用前，必须根据元素的自然顺序对数组进行升序排序（通过sort(Object[])方法）。

public static int[] copyOf(int[] original, int newLength)
	复制指定的数组。

public static <T> T[] copyOf(T[] original, int newLength)

public static int[] copyOfRange(int[] original, int from, int to)
public static <T> T[] copyOfRange(T[] original, int from, int to)

public static boolean deepEquals(Object[] a1, Object[] a2)
	如果两个数组彼此是深层相等的，则返回true。

public static boolean equals(Object[] a, Object[] a2)
	如果两个Object数组彼此相等，则返回true。

public static void fill(Object[] a, Object val)
public static void fill(Object[] a, int fromIndex, int toIndex, Object val)
	将val对象填充到Object数组指定范围。

public static void sort(int[] a)
public static void sort(int[] a, int fromIndex, int toIndex)
	对int型数组的指定范围按数字升序进行排序。

public static void sort(Object[] a)
public static void sort(Object[] a, int fromIndex, int toIndex)
	根据元素的自然顺序对指定对象数组的指定范围按升序进行排序。

public static String toString(Object[] a)
	返回指定数组内容的字符串表示形式。
