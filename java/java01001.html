第一节    Java语言概述


  • 一、Java语言的发展历史
  • 二、Java语言的主要特点
  • 三、Java语言的技术方向

一、发展历史

Java 语言是 Sun(Stanford University Network，斯坦福大学网络公司)1995年推出的一门高级编程语言。

1996年1月，Sun公司发布了Java的第一个开发工具包（JDK 1.0），标志着Java成为一种独立的开发工具。

1997年2月，JDK 1.1面世。

1998年12月8日，第二代Java平台的企业版J2EE发布。

1999年6月，Sun公司发布了第二代Java平台（简称为Java2）的3个版本：J2ME（Java2 Micro Edition，Java2平台的微型版），应用于移动、无线及有限资源的环境；J2SE（Java 2 Standard Edition，Java 2平台的标准版），应用于桌面环境；J2EE（Java 2Enterprise Edition，Java 2平台的企业版）。

2000年5月，JDK1.3、JDK1.4和J2SE1.3相继发布。

2001年9月24日，J2EE1.3发布。2002年2月26日，J2SE1.4发布。

2004年9月30日，J2SE1.5发布，更名为Java SE 5.0，包含了从1996年发布1.0版本以来的最重大的更新，其中包括泛型支持、基本类型的自动装箱、改进的循环、枚举类型、格式化I/O及可变参数。

2005年6月，Sun公司发布了Java SE 6。

2006年11月13日，Sun公司宣布将Java技术作为免费软件对外发布。

2009年，甲骨文公司宣布收购Sun。2011年，甲骨文公司发布Java7。

2014年，甲骨文公司发布Java8。

二、主要特点

Java吸收了C++的各种优点，摒弃了C++里难以理解的多继承、指针等概念。

面向Internet的编程语言，随着Java技术在web方面的不断成熟，已经成为web应用程序开发的首选语言。

面向对象的编程思想。从复杂的业务中抽象出通用的数据和业务逻辑，将这些数据和逻辑封装成类，通过类创建对象，对象再相互调用、组合形成模块化的功能组件，最后把许多个组件整合到一起就完成了整体的业务操作。

极好的跨平台性。JVM(Java Virtual Machine)实现了Java程序的跨平台特性。“一次编写，到处运行”。

高性能。较好的健壮性、动态性。

支持多线程。

简单易学，安全可靠。

三、技术方向

JavaEE(Java 2 Platform Enterprise Edition)企业版，是为开发企业环境下应用程序提供的一套解决方案，包含技术有Servlet、Jsp等，主要针对Web应用程序开发。

JavaSE(Java 2 Platform Standard Edition)标准版，开发普通桌面和商务应用程序。

JavaME(Java 2 Platform Micro Edition)小型版，开发电子消费产品和嵌入式设备。
