一、JDK与JRE

JDK：Java Development Kit，Java开发工具包，是给开发人员使用的，其中包含了Java的开发工具，如java、javac、jar等命令，同时也包含了JRE。

JRE：Java Runtime Environment，Java运行环境，包含了Java虚拟机和Java程序所需的核心类库。如果只是想运行一个开发好的Java程序，计算机只需要安装JRE就可以了。
