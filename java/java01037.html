一、Math 类

java.lang;

public final class Math extends Object {}

Math 类包含大量基本数学运算的方法，如初等指数、对数、平方根和三角函数。 

public static final double PI
	pi（圆周率）的double值。

public static int abs(int a)
	返回int值的绝对值。

public static double ceil(double a)
	返回大于等于参数的最小整数。

public static double floor(double a)
	返回小于等于参数的最大整数。

public static int max(int a, int b)
	返回两个int值中较大的一个。

public static int min(int a, int b)
	返回两个int值中较小的一个。

public static double pow(double a, double b)
	计算a的b次幂。

public static double random()
	返回一个伪随机double值，该值大于等于0.0 且小于1.0

	例子：
	double random1 = Math.random();
	System.out.println(random1);

	// 1 到 10 的随机数
	System.out.println((int)Math.ceil(random1 * 10));

	// 11 到 20 的随机数
	System.out.println((int)Math.ceil(random1 * 10 + 10));

public static long round(double a)
	四舍五入

二、Random 类

java.util;

public class Random extends Object implements Serializable {}

对于生成随机数这个问题，使用 Math 类的 random 方法不是很方便。

可以使用 Random 类的实例生成伪随机数。

实际开发中，有一些场合需要使用到随机数，比如坦克大战，坦克转向时需要一个随机数来确定方向，前进的长度也需要使用随机数来确定。再有一些需要使用概率随机抽奖的场景也可以使用随机数实现。 

public Random()
	创建一个新的随机数生成器。

public double nextDouble()
	返回下一个伪随机数，是在0.0和1.0之间均匀分布的double值。

public int nextInt()
	返回下一个伪随机数，是此随机数生成器序列中均匀分布的int值。

public int nextInt(int n)
	返回一个伪随机数，在0（包括）和n（不包括）之间均匀分布的int值。

	例子：
	Random r = new Random();

	int i = r.nextInt(10);
	System.out.println(i);
	System.out.println(i + 1);

三、Scanner 类

1、 Scanner 类 API
java.util;

public final class Scanner {}

如果开发一个 cmd 命令行程序，在程序运行过程中，可能需要用户输入一些参数，这种场景就需要使用 Scanner 类来读取命令行的输入信息。

Scanner 可以使用正则表达式来解析基本类型和字符串的简单文本扫描器。

Scanner 使用分隔符模式将其输入分解为标记，默认情况下该分隔符模式与空白匹配。然后可以使用不同的 next 方法将得到的标记转换为不同类型的值。

扫描操作可能被阻塞，而等待信息的输入。

Scanner 的多线程使用是不安全的。


构造方法 

public Scanner(File source) throws FileNotFoundException

public Scanner(File source, String charsetName) throws FileNotFoundException

public Scanner(InputStream source)

public Scanner(InputStream source, String charsetName)

public Scanner(Readable source)

public Scanner(String source)

主要方法 

public boolean hasNext()
	如果扫描器输入中有另一个标记，则返回true。

public boolean hasNext(Pattern pattern)
	如果下一个标记与模式匹配，则返回true。

public boolean hasNext(String pattern)
	如果下一个标记与字符串模式匹配，则返回true。

public boolean hasNextInt()
	如果通过nextInt()方法，扫描器输入信息中的下一个标记可以解释为默认基数中的一个int值，则返回true。

public boolean hasNextInt(int radix)
	如果使用nextInt()方法，扫描器输入信息中的下一个标记可以解释为指定基数中的一个int值，则返回true。

public boolean hasNextLine()
	扫描器的输入中存在另一行，则返回true。在等待输入信息时，此方法可能阻塞。

public String next()
	查找并返回此扫描器的下一个完整标记。

public int nextInt()
	将输入信息的下一个标记扫描为一个int。

public String nextLine()
	返回扫描器当前行。

2、 Scanner 类使用

读取控制台输入 

private static void test1() {

	// 创建读取控制台输入的扫描器
	Scanner scanner = new Scanner(System.in);

	int i = scanner.nextInt(); // 程序会阻塞
	System.out.println(i);

	scanner = new Scanner(System.in);

	String str = scanner.nextLine();
	System.out.println(str);
}

演示读取字符串 

private static void test2() {

	// 创建读取指定字符串的扫描器
	Scanner scanner = new Scanner("测试 Java 的 Scanner 类");

	while(scanner.hasNext()) {
		System.out.println(scanner.next());
	}
}

读取字符串nextLine 

private static void test3() {

	// 创建读取指定字符串的扫描器
	Scanner scanner = new Scanner("测试 Java 的 Scanner 类");

	while(scanner.hasNext()) {
		System.out.println(scanner.nextLine());
	}
}

读取字符串，自定义分隔符 

private static void test4() {
	String input = "1 fish 2 fish red fish blue fish";
	Scanner s = new Scanner(input).useDelimiter("\\s*fish\\s*");

	while(s.hasNext()) {
		System.out.println(s.next());
	}

	s.close();
}

读取文件 

private static void test5() throws FileNotFoundException {

	// 创建读取指定文件的扫描器
	Scanner s = new Scanner(new FileInputStream(
		new File("D:\\tmp\\java\\javanote\\src\\" + 
			"org\\net5ijy\\javanote\\api\\ScannerAPI.java")), "UTF-8");

	while(s.hasNextLine()) {
		System.out.println(s.nextLine());
	}

	s.close();
}
