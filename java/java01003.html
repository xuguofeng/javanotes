一、打开 Windows 命令行

第 1 种方法：开始菜单—>附件—>命令提示符可以打开cmd命令行

第 2 种方法：使用 Win+r 快捷键打开运行，输入cmd，可以打开cmd命令行

第 3 种方法：开始菜单—>运行，输入cmd，可以打开cmd命令行


二、Windows 常用命令

命令作用
cd [dirName]切换工作目录
cd ..返回上级目录
cd \返回根目录
dir列出当前目录下的文件和目录
md创建目录
rd删除目录
del删除文件
exit退出cmd

使用help命令可以查看帮助，例如：查看rd命令的使用方式，可以使用help rd即可列出该命令的一些参数。 

C:\>help rd
	删除一个目录。
	
	RMDIR [/S] [/Q] [drive:]path
	RD [/S] [/Q] [drive:]path
	
    /S      除目录本身外，还将删除指定目录下的所有子目录和
            文件。用于删除目录树。

    /Q      安静模式，带 /S 删除目录树时不要求确认
C:\>

Windows命令其实很多，比如常用的ping、netstat、telnet等，此处不多做介绍。

三、使用 java 命令

使用cd命令切换到 jdk 的 bin 目录下，即可使用java、javac等命令。 

C:\Users\XUGUOFENG>d:

D:\>cd "Program Files\Java6\jdk1.6.0_45"

D:\Program Files\Java6\jdk1.6.0_45>cd\

D:\>cd "Program Files\Java6\jdk1.6.0_45\bin"

D:\Program Files\Java6\jdk1.6.0_45\bin>java -version
	java version "1.6.0_45"
	Java(TM) SE Runtime Environment (build 1.6.0_45-b06)
	Java HotSpot(TM) Client VM (build 20.45-b01, mixed mode, sharing)

D:\Program Files\Java6\jdk1.6.0_45\bin>

例如：

java -version 用来查看java版本

javac 回车列出该命令使用方法

在不知道命令参数的情况下，直接输入命令回车可以列出命令使用帮助。

但是 java 的工具命令现在只能在切换到 bin 目录下才能使用，在其他目录下就无法使用。为了在任何目录下都可以使用 java 工具，我们需要配置系统环境变量。

四、配置环境变量

java 的工具命令只能在 bin 目录下才能使用。为了在任何目录下都可以使用 java 工具，我们需要配置系统环境变量。下面我们就讲解一下 Windows 操作系统下配置 java 环境变量的方法：

右键“计算机(XP系统是我的电脑)”，选择“属性”，选择“高级系统设置”


新建JAVA_HOME变量，编辑Path变量 

JAVA_HOME=D:\Program Files\Java6\jdk1.6.0_45

Path=%Path%;%JAVA_HOME%\bin

配置环境变量后就可以在任意目录下使用 java 工具命令

五、编写 QuickStart 程序

首先在 D 盘 tmp 目录下创建 java 目录，并创建 quickstart 项目 

C:\Users\XUGUOFENG>d:

D:\>cd tmp

D:\tmp>md java

D:\tmp>cd java

D:\tmp\java>md quickstart

可以得到如下目录结构

使用 cd 命令转到 quickstart 项目下，使用 notepad 命令创建 QuickStart.java 文件 

D:\tmp\java>cd quickstart

D:\tmp\java\quickstart>notepad QuickStart.java

在 QuickStart.java 文件中编写如下代码，并保存 

public class QuickStart{
	public static void main(String[] args){
		System.out.println("你好，Java");
	}
}

然后使用 javac 命令将 QuickStart.java 文件编译成 .class 字节码文件，使用 java 命令运行，注意不需要加 .class 文件后缀 

D:\tmp\java\quickstart>javac QuickStart.java

D:\tmp\java\quickstart>java QuickStart
你好，Java

如果您看到 cmd 命令行正确输出“你好，Java”，那么，恭喜您，您的第一个 java 程序就编写完成了
