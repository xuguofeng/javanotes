一、为什么需要循环结构

在控制台打印 5 次“Hello Java”，您最先想到的办法是什么？ 

public static void main(String[] args) {
	System.out.println("Hello Java");
	System.out.println("Hello Java");
	System.out.println("Hello Java");
	System.out.println("Hello Java");
	System.out.println("Hello Java");
}

这个确实是一个最简单、最易实现的办法，但是我如果想打印 10 次、20 次、100 次，怎么办？

再有，如果我们有一个几万行的 excel 文件，想逐行读取、封装对象进行数据库保存或其他的操作，该怎么办？

为了解决上面的两种需求，我们就需要一种可以处理大量相同或者相似操作的语法结构，即循环结构。

二、while 循环结构

/*
	判断 condition 条件为 true 时，会进入循环体代码块
	代码块执行之后，会再次执行上面的操作
	如此循环，直到 condition 条件为 false 才结束循环
*/
while(condition) {
	// 当condition条件表达式为true时，执行代码块
}

使用 while 循环解决一下上面打印 5 次“Hello Java”的需求 

public static void main(String[] args) {
	int times = 0; // 定义一个标识打印次数的变量
	/*
		 // 次数标识小于 5 时才进入循环体
	*/
	while (times < 5) {
		System.out.println("Hello Java");
		times++; // 次数标识变量自增
	}
}

我们对上面的代码做一下简单的说明 

int times = 0;

第一次循环判断：0 < 5
	打印 Hello Java
	times自增，times = 1
第二次循环判断：1 < 5
	打印 Hello Java
	times自增，times = 2
第三次循环判断：2 < 5
	打印 Hello Java
	times自增，times = 3
第四次循环判断：3 < 5
	打印 Hello Java
	times自增，times = 4
第五次循环判断：4 < 5
	打印 Hello Java
	times自增，times = 5
第六次循环判断：5 < 5 != true，跳出循环，打印了 5 次 Hello Java

在使用 while 循环时需要注意一下两点：

（1）循环结构一般都有结束循环的条件；

（2）循环所在代码块内、且在循环语句之前定义的变量，在循环体内可以使用，且不可以重复定义；

（3）循环体内定义的变量在本次循环执行完毕后释放；

下面我们来看两个 while 循环的例子

例 1：1 + 2 + …… + 100 = ? 

public static void main(String[] args) {
	int num = 1;
	int sum = 0;
	while ( num <= 100 ) {
		sum += num;
		num++;
	}
	System.out.println("1 + 2 + ...... + 100 = " + sum);
}

输出：

1 + 2 + ...... + 100 = 5050

例 2：1 至 100 中 7 的 倍数有几个？ 

public static void main(String[] args) {
	int count = 0;
	int num = 1;
	while ( num <= 100 ) {
		if ( num % 7 == 0 ) {
			count += 1;
		}
		num++;
	}
	System.out.println("1 至 100 中 7 的 倍数有：" + count);
}

输出：

1 至 100 中 7 的 倍数有：14

四、for 循环结构

基本语法 

for (初始化表达式; 条件判断表达式; 循环后执行表达式) {
	// 执行代码；
}

具体特点

（1）初始化表达式只执行一次，定义的变量在整个循环过程中都有效；

（2）条件判断表达式每次循环开始时进行判断，为 true 时进入代码块，为 false 时结束循环；

（3）循环后执行的表达式通常用于控制循环条件；

（4）循环体内定义的变量在本次循环执行完毕后释放


使用嵌套 for 循环打印九九乘法表 

for (int i = 1; i <= 9; i++) {
	for (int j = 1;j <= i; j++) {
		System.out.print(i+" x "+j+" = "+(i*j)+"\t");
	}
	System.out.println();
}

for 和 while 循环的特点及区别

（1）二者可以相互替换；

（2）for 循环中初始化表达式定义的变量只在 for 语句内部可以使用，循环结束变量释放；如果使用变量的值来控制 while，这个变量需要在之前定义，while 循环结束后还可以使用；


各种流程控制的结构可以互相嵌套，上面打印乘法表就属于嵌套。

下面我们把内层循环改造成 while 循环结构 

for (int i = 1; i < 10; i++ ) {

	int j = 1;

	while (true) {
		System.out.print(i + " x " + j + " = " + (i * j) + "\t");
		j++;

		/*
			主要是演示结构嵌套，不要这样使用，而是在 while 中使用 j <= i 表达式来控制循环跳出。
		*/
		if (j > i) // 这段if代码可以不使用
			break; // break关键字可以跳出循环
	}
	System.out.println(); // 打印个换行
}

六、break 和 continue 关键字

break 关键字：作用是跳出 switch 和循环语句。

continue 关键字：作用在循环语句中，用于跳过本次循环。


一个简单的例子即可说明这个问题。 

for (int i = 0; i < 10; i++) {
	if (i % 2 == 0)  // i为2的倍数时，跳过本次循环
		continue;
	
	if (i > 5) 
		break; // i大于5时，结束循环
	
	System.out.println(i);
}

输出：

1
3
5

演示代码在项目 ProcessControll 中的 ContinueAndBreak.java 文件。

七、给循环起“名字”

break 和 continue 关键字可以结束循环和跳过本次循环，但操作的是关键字所在的循环语句，无法操作嵌套在此循环外面的循环。

我们可以给循环命名，然后在循环内部使用“break 循环名字”、“continue 循环名字”方式解决上面的问题，格式是这样的： 

LoopName1:for(表达式1; 表达式2; 表达式3) {
	// 循环体;
	
	LoopName2:while(condition) {
		// 循环体;

		if (condition) {
			continue LoopName2; // 跳过本次内层循环
		}

		if (condition) {
			break LoopName1; // 结束外层循环
		}
	}
}
