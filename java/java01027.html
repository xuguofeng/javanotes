第二十七节    基本数据类型包装类


  • 一、概述

一、概述

包装类就是将基本数据类型封装成类，因为基本数据类型其实不是很符合面向对象的思想，而在类中可以定义更多的功能方法操作数据，比如基本数据类型与字符串之间的转换。

基本数据类型包装类型
byteByte
shortShort
intInteger
longLong
doubleDouble
floatFloat
charCharacter
booleanBoolean

在此以 Integer 类举例。

二、Integer 类介绍

java.lang;

public final class Integer extends Number implements Comparable<Integer> {}

Integer 类在对象中包装了一个基本类型 int 的值。

该类提供了多个方法，能在 int 类型和 String 类型之间互相转换，还提供了处理 int 类型时非常有用的一些常量和方法。

三、Integer 字段

public static final int MAX_VALUE
	int类型数据的最大值。

	例：
	int max = Integer.MAX_VALUE;
	System.out.println(max);  // 2147483647

public static final int MIN_VALUE
	int类型数据的最小值。

四、Integer 类构造方法

public Integer(int value)
	构造一个新的Integer对象，有指定的int值。

public Integer(String s) throws NumberFormatException
	构造一个新的Integer对象，表示String参数所指示的int值。

	例：
	Integer a1 = new Integer(1);
	System.out.println(a1); // 1

	Integer a2 = new Integer("2");
	System.out.println(a2); // 2

五、Integer 类主要方法

public int compareTo(Integer anotherInteger)
	在数字上比较两个Integer对象。

	例：
	Integer a1 = 1;
	Integer a2 = 5;
	System.out.println( a1.compareTo(a2) ); // -1

public int intValue()
	以int类型返回该Integer的值。

public static int parseInt(String s) throws NumberFormatException
	将字符串参数作为有符号的十进制整数进行解析。

public static int parseInt(String s, int radix) throws NumberFormatException
	使用第二个参数指定的基数，将字符串参数解析为有符号的整数。

	例：
	String str = "15";
	Integer a3 = Integer.parseInt( str );
	Integer a4 = Integer.parseInt( str, 16 );
	System.out.println( a3 ); // 15
	System.out.println( a4 ); // 21

public static int signum(int i)
	返回指定int值的符号函数。如果值为负，则返回-1；如果值为零，则返回0；如果值为正，则返回1。

	例：
	System.out.println( Integer.signum(10) );  // 1
	System.out.println( Integer.signum(-10) ); // -1
	System.out.println( Integer.signum(0) );   // 0

public static String toBinaryString(int i)
	二进制字符串

public static String toHexString(int i)
	十六进制字符串

public static String toOctalString(int i)
	八进制字符串

	例：
	Integer a1 = 153;

	String binary = Integer.toBinaryString(a1);
	String hex = Integer.toHexString(a1);
	String octal = Integer.toOctalString(a1);

	System.out.println( binary ); // 10011001
	System.out.println( hex );    // 99
	System.out.println( octal );  // 231

public static String toString(int i, int radix)
	字符串表示形式，第二个参数指定基数
        
	例如：
	Integer.toString(27, 16).toUpperCase();

public static Integer valueOf(int i)
	返回一个表示指定的int值的Integer实例

public static Integer valueOf(String s) throws NumberFormatException
	返回保存指定String值的Integer对象

public static Integer valueOf(String s, int radix) throws NumberFormatException
	返回保存指定String值的Integer对象，第二个参数提供的基数

	例：
	Integer a1 = Integer.valueOf( 153 );
	Integer a2 = Integer.valueOf( "153" );
	Integer a3 = Integer.valueOf( "153", 16 );

	System.out.println(a1); // 153
	System.out.println(a2); // 153
	System.out.println(a3); // 339
