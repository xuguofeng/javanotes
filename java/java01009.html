一、方法概述

方法就是定义在类中、能够处理业务逻辑的程序代码，也称为函数。

定义格式如下： 

[修饰符] [static] [final] [abstract] 返回值类型 方法名( 参数类型 参数1, 参数类型 参数2, ...... ){
	执行代码;
	return 返回值;
}

方法的一些特点和好处：

（1）方法就是对一些通用业务功能的封装，可以提高代码的重用性和扩展性，体现着面向对象的编程思想；

（2）方法被调用才会执行；

（3）对于没有返回值的方法，需要使用关键字void来声明，return可以省略；

（4）不可以在方法中定义方法；

（5）驼峰命名法，多个单词时，第一个单词的首字母小写，其余单词首字母大写；

（6）方法的定义需要注意分析具体的业务需求，从需求出发，确定方法需要的参数、返回值类型，切忌为了使用方法而定义方法。

二、方法的定义和调用

下面我们使用几个简单的例子演示下方法的定义和调用。

例子1：使用 * 画 5 x 5 的矩形 

/*
	使用 * 画 5 x 5 的矩形
*/
public static void printRect() {
	for (int i = 0; i < 5; i++) {
		for (int j = 0; j < 5;j++) {
			System.out.print("*");
		}
		System.out.println();
	}
}

public static void main(String[] args) {
	/*
		调用方法
		如果方法有参数，需要在括号中传入参数
	*/
	printRect();
}

输出：

*****
*****
*****
*****
*****

例子2：找出两个整数中较大的一个 

/*
	返回a、b中较大的一个，参数有两个，返回值 int 类型
*/
public static int getMax(int a, int b){
	return a >= b ? a : b; // 三元运算
}

public static void main(String[] args) {
	int a = getMax(1, -1); // 使用 int 变量接收方法返回值
	System.out.println(a);
}

输出：

1

三、方法的重载

方法重载就是在同一个类中，允许存在多个同名方法，前提是它们的参数个数或参数类型不同。java 依据方法的参数列表来判断调用的是哪个方法，与方法返回值无关，所以方法可以重载。

我们之前编写的打印 5 x 5 矩形方法，虽然可以实现我们的需求，但是如果我想打印 6 x 6 的矩形呢？难道再定义一个方法吗？所以，这个方法需要重载一个可以指定行、列的方法。 

/*
	打印5x5的矩形
*/
public static void printRect() {
	printRect(5,5);
}

/*
	打印指定长、宽的矩形
*/
public static void printRect(int rows,int cols) {
	int rowsTmp = 5; // 默认是 5 行
	int colsTmp = 5; // 默认是 5 列
	if (rows > 0) rowsTmp = rows;
	if (cols > 0) colsTmp = cols;
	for (int i = 0; i < rowsTmp; i++) {
		for (int j = 0; j < colsTmp; j++) {
			System.out.print("*");
		}
		System.out.println();
	}
}

public static void main(String[] args) {
	printRect(); // 调用 5 x 5 方法
	System.out.println("---------------------------------");
	printRect(8,9);
}

这是一个很有意义的例子，因为在实际开发过程中，经常是在重载方法中调用另外的同名方法。


下面我们再改造一下九九乘法表的代码，重载一个可以指定行数的方法： 

/*
	可以指定打印的行数
*/
public static void multiplicationTable(int rows) {
	if(rows <= 0 || rows > 9) rows = 9;
	for (int i = 1; i <= rows; i++) {
		for (int j = 1; j <= i; j++) {
			System.out.print(j+" x "+i+" = "+(i*j)+"\t");
		}
		System.out.println();
	}
}

/*
	实际使用中我们打印 9 x 9 的乘法表的场景更多，如果使用此方法，每一次都要传入参数，所以我们可以重载一个默认的方法
*/
public static void multiplicationTable(){
	multiplicationTable(9); // 调用另外一个可以传参的方法，参数为 9
}

public static void main(String[] args) {
	multiplicationTable();
	System.out.println("---------------------------------");
	multiplicationTable(4);
}

四、main 方法的运行过程

使用 java 命令启动 jvm 运行程序时，会加载字节码文件，在里面找 main 方法，找不到会出现异常：

找到 main 方法之后，将 main 方法放入方法区，开始执行 main 方法。

在执行 main 方法过程中，如果调用其他方法，就会把该方法放入方法区，执行该方法之后，jvm 会将其从内存中清除，释放内存，继续向下执行 main 方法。main 方法执行完毕后，程序结束，jvm 关闭。
