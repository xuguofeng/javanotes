二、算术运算符

1、 基础算术运算符

运算符功能
+加，还可以用于字符串的连接
-减
*乘
/除
%取余

+ 作为字符串连接符号 

String name = "xugf";
int age = 26;
System.out.println(name + " 年龄是 " + age + "。");

输出：

xugf 年龄是 26。

int 类型相除会把小数丢弃，只留下整数部分。如果需要保留小数，可以将类型转换为 float 或 double 

int a = 43;
System.out.println(a / 10);
System.out.println(a / 10f); // 将结果转为 float 类型
System.out.println(a / 10d); // 将结果转为 double 类型

输出：

4
4.3
4.3
2、 自增自减运算符

运算符功能
a+++1 运算前保留原值
a---1 运算前保留原值
++a+1 运算不保留原值
--a-1 运算不保留原值 

// a++
int a1 = 1;
int b1 = a1++;
System.out.println(a1); // 2
System.out.println(b1); // 1

// ++a
a1 = 1;
b1 = ++a1;
System.out.println(a1); // 2
System.out.println(b1); // 2

输出：

2
1
2
2

int b = a++ 的过程是这样的：a 需要参与赋值运算，所以 a 在做自增运算前会把原值保留，再进行自增运算，此时 a 的值为自增后的值，然后会把保留的原值赋值给 b

b = ++a 的过程是 a 自增，而不会保留原值，直接使用新值


a 变量为 int 类型，值为 1 ，让 a 变量增加 2，该怎么办？ 

int a = 1;
a + 2;

上面的代码不是很正确，a 确实和 2 相加了，而且确实也得到了一个值为 3 的 int 型数据，但是如果我们打印下 a，会发现 a 的值还是 1，原因是 a + 2 的结果实际上没有赋值给 a，需要做下改正 

int a = 1;
a = a + 2; // 使用 a 接收相加的结果

下面我们介绍一种运算符可以让上面的代码变得更加简洁

运算符功能
+=将变量加某个值再赋值给该变量
-=将变量减某个值再赋值给该变量
*=将变量乘某个值再赋值给该变量
/=将变量除某个值再赋值给该变量
%=将变量取余某个值再赋值给该变量

上面 a + 2 的代码可以这样写 

int a = 1;
a += 2; // 把 a 与 2 相加，把结果再赋值给 a

这几种运算符在进行计算时会进行类型的检测转换操作，所以如下代码可以正常编译执行 

short a = 1;
a += 2; // 在做赋值操作时编译器会进行结果检测，如果结果在 short 型范围，会进行类型转换

四、逻辑运算符

用于连接两个 boolean 类型表达式，返回值还是一个 boolean 类型的值

运算符功能
&与，两边都为真返回真
&&短路与，两边都为真返回真，前为假直接返回假
|或，两边一个为真返回真
||短路或，两边一个为真返回真，前为真直接返回真
<=小于等于
!非，!true = false
^异或，两边只有一个为真才为真，了解即可

在这里我们需要详细讲解一下 && 和 ||

&& 两边的 boolean 表达式都为真返回真，但是和 & 不同的是，&& 在前面表达式为假时就直接返回假，而不去判断第二个表达式的真假，下面代码可以说明这个问题。为了演示这个效果，在类中添加两个方法（关于“方法”后面有详细讲解） 

public static void main(String[] args) {
	// 演示&&
	test3();
}
private static void test3() {
	System.out.println(m1() &  m2());
	System.out.println(m1() && m2());
}
private static boolean m1() {
	System.out.println("m1方法调用");
	return false;
}
private static boolean m2() {
	System.out.println("m2方法调用");
	return true;
}

输出：

m1方法调用
m2方法调用
false
m1方法调用
false

从代码输出可以看出，& 把两个表达式都判断了，而 && 只判断了前面的表达式

|| 两边的 boolean 表达式有一个为真就返回真，但是和 | 不同的是，|| 在前面表达式为真时就直接返回真，而不去判断第二个表达式的真假，下面代码可以说明这个问题 

public static void main(String[] args) {
	// 演示||
	test4();
}
private static void test4() {
	System.out.println(m3() |  m4());
	System.out.println(m3() || m4());
}
private static boolean m3() {
	System.out.println("m3方法调用");
	return true;
}
private static boolean m4() {
	System.out.println("m4方法调用");
	return false;
}

输出：

m3方法调用
m4方法调用
true
m3方法调用
true

五、位运算

运算符功能举例
<<左移3<<2 = 12 等同 3 * 2 * 2 = 12
>>右移3>>1 = 1 等同 3/2 = 1
>>>无符号右移3>>>1 = 1 等同 3/2 = 1
&与运算6 & 3 = 2
|或运算6 | 3 = 7
^异或运算6 ^ 3 = 5
~反码~6 = -7

<<左移运算符，高位去掉，低位补0 

int a = 6;
a = a << 1;
System.out.println(a);

输出：

12

>>右移运算符，高位按照原来的数字进行补充，低位去掉。>>>效果类似，只是高位使用 0 补充，低位去掉 

int a = 24;
a = a >> 1;
System.out.println(a);

输出：

12

& 与运算符，两个为 1 才为 1，有一个为 0 就是 0 

int a = 24
a = a & 10;
System.out.println(a);

输出：

12

| 或运算符，两个有一个为 1 即为 1 

int a = 24;
a = a | 10;
System.out.println(a);

输出：

26

^ 异或运算符，两个只有在不同时才为 1 

int a = 24;
a = a ^ 10;
System.out.println(a);

a = a ^ 10;
System.out.println(a);

输出：

18
24

~ 取反运算符 

int a = 24;
a = ~a;
System.out.println(a);

输出：

-25

六、三元运算符

条件表达式 ? 返回值 1 : 返回值 2 

int x = 2;
int y = 3;

int z = x > y ? x : y;

System.out.println(z);

输出：

3
