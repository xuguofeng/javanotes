百度搜索 “plsql developer”

找到 “PL/SQL Developer 10.0 page”

二、安装

三、instantclient 下载安装

1、 下载

百度搜索 “instantclient”

找到 “Downloads”

选择 “Instant Client for Microsoft Windows (x64)”

找到 “instantclient-basic-windows.x64-11.2.0.4.0.zip”

登陆后下载即可

2、 解压安装

把解压后的 “instantclient_11_2” 拷贝到 “C:\Program Files” 下面

3、 配置PLSQL Developer

打开软件，找到 Configure—>Preference，找到 Connection 菜单，配置如下

Oracle Home 

C:\Program Files\instantclient_11_2

OCI library 

C:\Program Files\instantclient_11_2\oci.dll

关闭 PLSQL Developer

4、 配置tns

找到 OracleXE 安装目录下的 tnsnames.ora 配置文件，在以下目录： 

C:\soft01\oracle\app\oracle\product\11.2.0\server\network\ADMIN

把 tnsnames.ora 拷贝到 “C:\Program Files\instantclient_11_2\NETWORK\ADMIN”，如果没有这个目录需要创建

四、连接数据库

1、 连接本地数据库

打开 PLSQL Developer

Database选择 XE，Username是 sys，Password是 Sysmanager1，Connect as选择 SYSDBA

2、 连接远程数据库

如果您需要连接远程数据库，首先必须有远程数据库的tns信息，把这些信息追加到 C:\Program Files\instantclient_11_2\NETWORK\ADMIN\tnsnames.ora 文件，这样再次打开 PLSQL Developer 即可连接这个远程的 Oracle 服务器了

五、普通用户登陆数据库

1、 创建用户

打开“命令窗口”
