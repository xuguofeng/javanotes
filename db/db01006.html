一、环境概述

操作系统：64 位 Windows 7 操作系统


MySQL 版本：mysql-5.5.59-winx64.msi

二、MySQL 下载

百度搜索 “mysql下载”


找到 “社区版本”



选择 “MySQL Community Server 5.5”


选择 “Windows(x86, 64-bit), MSI Installer”

三、MySQL 安装



选择 “Custom”


这步可以选择安装目录





打开 “配置向导”


选择 “Detailed Configuration”


选择 “Developer Machine”


选择 “Multifunctional Database”




可以看到MySQL服务监听3306端口，勾选防火墙放行


选择 “utf8” 编码


选择系统服务名


设置root用户密码，禁用root用户远程连接



四、MySQL 系统服务

我们首先打开 cmd 命令行，看一下 MySQL 是否启动。


可以看到服务已经启动了。

MySQL 在安装时，默认注册系统服务，而且也是开机自动启动的。我们可以根据自己的需要修改服务为手动启动。


我们可以在 “服务” 管理中对 MySQL 进行启动、关闭操作。

我们还可以自己编写 .bat 脚本对 MySQL 进行启动、关闭操作。

启动脚本：MySQLStart.bat 

net start MySQL
pause

关闭脚本：MySQLStop.bat 

net stop MySQL
pause

使用管理员权限执行脚本即可。

五、命令行连接 MySQL

首先需要确认 path 环境变量有 mysql 命令的路径，如果没有需要配置

计算机—>属性—>高级系统设置—>环境变量


打开cmd命令行 

C:\Users\XU>mysql -uroot -p123456

查看下当前的用户信息 

mysql> use mysql;
Database changed
mysql> select user, host, password from mysql.user;
+------+-----------+------------------------------------+
| user | host      | password                                  |
+------+-----------+------------------------------------+
| root | localhost | *6BB4837EB74329105EE4568DDA7DC67ED2CA2AD9|
+------+-----------+-------------------------------------+
1 row in set (0.00 sec)

我们使用局域网的另外一台服务器（192.168.0.10）连接 mysql 服务，发现连接被拒绝 

C:\Users\XGF>mysql -uroot -p123456 -h 192.168.0.102
ERROR 1130 (HY000): Host 'TEST-PC' is not allowed to connect to this MySQL server

这是因为在安装 MySQL 时禁止了 root 用户的远程连接，所以我们先创建一个用户用于局域网连接管理 

mysql> grant all on *.* to 'system'@'localhost' identified by '123456';
Query OK, 0 rows affected (0.00 sec)

mysql> grant all on *.* to 'system'@'192.168.0.%' identified by '123456';
Query OK, 0 rows affected (0.00 sec)

mysql> flush privileges;
Query OK, 0 rows affected (0.00 sec)

现在，再回到 192.168.0.10 上来，使用 system 用户连接 

C:\Users\XGF>mysql -usystem -p123456 -h 192.168.0.102

至于 mysql 相关命令的使用在此就不做过多的介绍了，mysql 有非常详细的官方文档可以参考。
